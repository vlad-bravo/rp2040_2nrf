Приёмник конфигурируется на автоотправку payload

Передатчик отправляет посылку
В течение некоторого времени проверяет status, бит tx_ds
Если бит установлен - отправка прошла успешно, есть подтверждение
Можно проверить в status бит rx_dr
Если установлен - можно считывать payload

Приёмник приняв командную посылку должен её обработать
Передать на компьютер, мы будем видеть, что командные пакеты приходят

Когда приёмник получает новый payload, он просто загружает cmd_w_ask_payload
